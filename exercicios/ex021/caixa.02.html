Grouping tags

Tags de agrupamento são usadas para agrupar outros elementos HTML. Elas não possuem um significado semântico específico, mas são úteis para aplicar estilos ou scripts a um conjunto de elementos. As tags de agrupamento mais comuns são <div> e <span>.

Exemplo com <div>

Conteúdo dentro de uma div

Este parágrafo está dentro de uma tag <div>. A tag <div> é um elemento de nível bloco (block-level), o que significa que ela ocupa toda a largura disponível e começa em uma nova linha.

Exemplo com <span>

Este é um parágrafo com um texto destacado usando a tag <span>. A tag <span> é um elemento inline (inline-level), o que significa que ela não inicia uma nova linha e só ocupa o espaço necessário para seu conteúdo.
